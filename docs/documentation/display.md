# Display

We can set the display property as follows:

- `.display-block`
- `.display-inline-block`
- `.display-inline`
- `.display-table`
- `.display-table-row`
- `.display-table-cell`
- `.display-none`

Note: Flex display in a different section

Also, we can use the breakpoints prefixes, i.e:

- `.sm:display-none`
- `.md:display-block`
- `.lg:display-inline`
- `.xl:display-table`
