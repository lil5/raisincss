# Padding

- `.p-0` -> `padding: 0rem`
- `.p-1` -> `padding: 0.25rem`
- `.p-2` -> `padding: 0.5rem`
- `.p-3` -> `padding: 0.75rem`
- `.p-4` -> `padding: 1rem`
- `.p-5` -> `padding: 1.25rem`
- `.p-6` -> `padding: 1.5rem`
- `.p-7` -> `padding: 1.75rem`
- `.p-8` -> `padding: 2rem`
- `.p-9` -> `padding: 2.25rem`
- `.p-10` -> `padding: 2.5rem`
- `.p-11` -> `padding: 2.75rem`
- `.p-12` -> `padding: 3rem`

We can also use top, right, bottom and left paddings:

- `.pt-{0 to 12}` -> `padding-top: {0 to 3}rem`
- `.pr-{0 to 12}` -> `padding-right: {0 to 3}rem`
- `.pb-{0 to 12}` -> `padding-bottom: {0 to 3}rem`
- `.pl-{0 to 12}` -> `padding-left: {0 to 3}rem`

Or use the x and y paddings:

- `.px-{0 to 12}`
- `.py-{0 to 12}`

And we can use any breakpoint prefix, i.e:

- `.sm:p-4`
- `.md:pb-12`
- `.lg:px-2`
- `.xl:pr-0`
