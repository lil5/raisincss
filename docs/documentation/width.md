# Width

- `.w-0` -> `width: 0`
- `.w-1/2` -> `width: 50%`
- `.w-1/3` -> `width: 33.33333%`
- `.w-2/3` -> `width: 66.66667%`
- `.w-1/4` -> `width: 25%`
- `.w-3/4` -> `width: 75%`
- `.w-1/5` -> `width: 20%`
- `.w-2/5` -> `width: 40%`
- `.w-3/5` -> `width: 60%`
- `.w-4/5` -> `width: 80%`
- `.w-1/6` -> `width: 16.66667%`
- `.w-full` -> `width: 100%`
- `.w-screen` -> `width: 100vw`
- `.w-auto` -> `width: auto`

or

- `.sm:w-1/2`
- `.md:w-3/4`
- `.lg:w-full`
- `.xl:w-auto`
