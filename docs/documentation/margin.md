# Margin

- `.m-0` -> `margin: 0rem`
- `.m-1` -> `margin: 0.25rem`
- `.m-2` -> `margin: 0.5rem`
- `.m-3` -> `margin: 0.75rem`
- `.m-4` -> `margin: 1rem`
- `.m-5` -> `margin: 1.25rem`
- `.m-6` -> `margin: 1.5rem`
- `.m-7` -> `margin: 1.75rem`
- `.m-8` -> `margin: 2rem`
- `.m-9` -> `margin: 2.25rem`
- `.m-10` -> `margin: 2.5rem`
- `.m-11` -> `margin: 2.75rem`
- `.m-12` -> `margin: 3rem`

We can also use top, right, bottom and left margins:

- `.mt-{0 to 12}` -> `margin-top: {0 to 3}rem`
- `.mr-{0 to 12}` -> `margin-right: {0 to 3}rem`
- `.mb-{0 to 12}` -> `margin-bottom: {0 to 3}rem`
- `.ml-{0 to 12}` -> `margin-left: {0 to 3}rem`

Or use the x and y margins:

- `.mx-{0 to 12}`
- `.my-{0 to 12}`
- `.mx-auto`
- `.my-auto`

And we can use any breakpoint prefix, i.e:

- `.sm:m-4`
- `.md:mb-12`
- `.lg:mx-auto`
- `.xl:mr-0`
