# Height

- `.h-0` -> `height: 0`
- `.h-1/2` -> `height: 50%`
- `.h-1/3` -> `height: 33.33333%`
- `.h-2/3` -> `height: 66.66667%`
- `.h-1/4` -> `height: 25%`
- `.h-3/4` -> `height: 75%`
- `.h-1/5` -> `height: 20%`
- `.h-2/5` -> `height: 40%`
- `.h-3/5` -> `height: 60%`
- `.h-4/5` -> `height: 80%`
- `.h-1/6` -> `height: 16.66667%`
- `.h-full` -> `height: 100%`
- `.h-screen` -> `height: 100vh`
- `.h-auto` -> `height: auto`

or

- `.sm:h-1/2`
- `.md:h-3/4`
- `.lg:h-full`
- `.xl:h-auto`
